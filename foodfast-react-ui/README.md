**Tính năng:**
- Dự án có hai giao diện: giao diện người dùng và giao diện quản trị.
- Người dùng có thể đặt món ăn và thiết lập số lượng theo yêu cầu.
- Người dùng cũng có thể xem thông tin hồ sơ và lịch sử đơn hàng.
- Người dùng có thể thay đổi thông tin liên hệ cơ bản cũng như mật khẩu.
- Quản trị viên có thể thực hiện hai tác vụ:
- Thêm món ăn mới vào thực đơn.
- Thay đổi giá món ăn theo tỷ lệ lạm phát.
- Xem và xử lý đơn hàng của khách hàng.
- Quản trị viên cũng có thể tự mình kiểm tra hoạt động của trang web.
- Sử dụng cookie để đăng nhập vào tài khoản.

## Cài đặt và chạy:
- Clone the repository hoặc tải xuống thư mục zip.
- Giải nén thư mục zip.
- Nhập dữ liệu từ tệp ``` FoodOrderingWebsite.sql ``` vào Cơ sở dữ liệu ``` MySQL``` của bạn.
- Cài đặt NodeJS vào hệ thống của bạn.
- Vào thư mục bạn đã giải nén hoặc sao chép dự án.
- Mở dấu nhắc lệnh hoặc terminal có cùng vị trí với dự án của bạn.
- Gõ ``` npm install ``` trong dấu nhắc lệnh hoặc terminal.
- Bây giờ để chạy dự án trên máy chủ, hãy gõ ``` npm start ```.
